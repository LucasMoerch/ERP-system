This Enterprise Resource Planning (ERP) system was developed as part of **Project 3 at Aalborg University Copenhagen (AAU CPH)** by **Group 9**.

## Prerequisites
- Docker & Docker Compose
- Make

## Development Setup

To build and start the development environment, run the following command from the project root:

```bash
make dev
```
To stop the development containers:
```
make dev-down
```

## Environment Variables
Environment variables are defined in .env files and are used for configuration such as database connections and authentication.
